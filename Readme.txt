Configurations
1. Change the “api_address” string value in “strings.xml” to your API address like “http://localhost:8080/api/v1/%1$s”.
2. Update SDK tools(Google Play Services is mandatory) If you are executing in Android emulator.
3. Install any dependencies of nodejs to start the mock API.
4. Update Android SDK version if needed.
